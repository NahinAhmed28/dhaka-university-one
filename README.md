# Dhaka University One

Dhaka University One is a university-focused web application project.

## Features

- University workflow or portal foundation
- Public or administrative information screens
- Database/API integration potential
- Versioned project phase for Dhaka University work

## Modules

- Portal module: public pages, navigation, and content
- Workflow module: forms, submissions, or service actions when implemented
- Admin module: management screens and configuration
- Data module: database records or API responses
- Asset module: styles, scripts, and media resources

## System Architecture

The project should be organized as a web portal/application. The presentation layer renders university workflows, controllers/services coordinate actions, and a data layer stores or retrieves records. It can be extended with admin and reporting modules.

## Getting Started

```bash
git clone https://github.com/NahinAhmed28/dhaka-university-one.git
cd dhaka-university-one
composer install
npm install
npm run dev
```
